# Textfield

#### `Textfield` gir brukere muligheten til å skrive fritekst eller tall.

<Primary />
<Controls />

---

## Bruk

`Textfield` er bygd på [`Field`](/docs/komponenter-field--docs) og kan rendre som `Input` eller `Textarea` via `multiline`. Ikke pakk `Textfield` inn i `Field` – det er allerede inkludert.

```tsx
import { Textfield } from 'rk-designsystem';

<Textfield label="Navn" />
```

## Retningslinjer

Bruk `Input` for korte svar og `Textarea` for lengre fritekst. Unngå `placeholder` som instruks – den forsvinner ved skriving og må møte kontrastkrav. Tilpass bredde etter forventet innhold. Tillat kopier/lim inn.

### Inndata
Velg passende `type` og bruk `autocomplete` der formål finnes i WCAG‑listen. For rene tallfelt, foretrekk `type="text"` med `inputmode="numeric"` over `type="number"` for bedre robusthet.

---

## Tekst

Alltid label. Hold den kort (helst én linje), og unngå kolon.

---

## Tilgjengelighet

Unngå deaktivert (`disabled`) der det er mulig; vis heller tekst eller bruk `readOnly`. Prefiks/suffiks er dekorative og må suppleres av en beskrivende label.
