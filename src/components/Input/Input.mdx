# Input

#### `Input` er et grunnleggende skjemafelt for innsamling av brukerdata. Bruk den når du trenger enkel, fleksibel kontroll over oppsett og validering – for eksempel når du bygger spesialtilpassede felt.

#### Se også [`Textfield`](/docs/komponenter-textfield--docs), som kombinerer `Label`, beskrivelse og validering for deg.

## Retningslinjer

Se retningslinjer for [`Textfield`](/docs/komponenter-textfield--docs) for anbefalt plassering av ledetekst og beskrivelse. Ledeteksten bør stå over feltet for best lesbarhet – spesielt på små skjermer og når feilmeldinger vises.

Når ledetekst skjules visuelt (for eksempel i tabeller), må den fortsatt være tilgjengelig for skjermlesere via `aria-label` eller `aria-labelledby`.

---

## Tilgjengelighet

Koble alltid feltet med en `Label` ved hjelp av `htmlFor`/`id` eller `aria-labelledby`. Ved feiltilstand, sett `aria-invalid` og koble feilmeldingen via `aria-describedby` om mulig.
