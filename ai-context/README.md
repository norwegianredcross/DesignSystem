# AI Context Pipeline

This directory documents the machine-readable context published alongside the human-readable AI guide and component metadata.

## Published Outputs

- `AI_DESIGN_SYSTEM_GUIDE.md`
- `metadata.json`
- `ai-context.manifest.json`

## Intent

- Keep the checked-in guide useful for engineers and AI-assisted workflows.
- Publish a structured manifest so downstream repos can refresh local rules and detect drift without scraping a giant prose document.
- Preserve the existing guide-generation flow unless a safer refactor is clearly justified.
